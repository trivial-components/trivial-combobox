Configuration Options

        

            

                
                

                    Option
                    Description
                    Default
                

                
                
                

                    valueProperty
                    Name of the property which should be submitted on form submit. null is not allowed.
                    
                    allowFreeText ? 'displayValue' : 'id'
                

                

                    distinct
                    Determines whether duplicate entries are allowed. Duplicate entries are identified by their valueProperty property.
                    true
                

                

                    valueSeparator
                    String used to separate entry identifiers in the original input field.
                    
                    ','
                

                

                    allowFreeText
                    
                        Allow free text input.

                        If true, the user is not forced to select one of the suggested entries but can enter any tags he wants. Fake entries will be generated with the inputTextProperty field set to the value entered by the user.

                        If false, the user input will be deleted if he does not select a suggested entry.

                    		
                    true
                

                

                    freeTextSeparators
                    Array of separator chars the user may enter to separate free text tags.
                        Only evaluated if allowFreeText == true.
                    [',', ';']
                

                

                    inputTextProperty
                    Name of the property which will be filled into the input field when the user selects an entry. This property is also used for auto-completion.
                    'displayValue'
                

                

                    template
                    Mustache template to be used for displaying the tagbox dropdown entries.
                    TrivialComponents.icon2LinesTemplate
                

                

                    selectedEntryTemplate
                    Mustacheustache template to be used for displaying the selected tags.
                    Same as template, but wrapped with a tag display.
                

                

                    spinnerTemplate
                    Html code for the "loading..." indicator.
                    defaultSpinnerTemplate
                

                

                    noEntriesTemplate
                    Html code for indicating that there is no entry in the dropdown.
                    defaultNoEntriesTemplate
                

                

                    entries
                    The dropdown entries to be displayed initially.
                    []
                

                

                    selectedEntry
                    The initially selected entry.
                    undefined
                

                

                    queryFunction
                    JavaScript function for retrieving entries depending on the user's input. Takes two parameters:
                        queryString (user input) and
                        resultCallback (should be called with the list retrieved entries).
                    
                    Simple client-side implementation that filters the initially configured array of entries.
                

                

                    autoComplete
                    If true, the component user get auto-completion inside the text field.
                    true
                

                

                    autoCompleteDelay
                    Delay for auto-completion. (This feature may be removed, as there is currently no reason for a delay in auto-completion.)
                    0
                

                

                    showTrigger
                    The trigger is the button-like handle on the side of the tagbox. If false: no trigger is shown.
                    true
                

                

                    matchingOptions
                    An object that defines how the entries in the dropdown should be matched and highlighted. Properties are matchingMode, ignoreCase and maxLevenshteinDistance, 

                        Possible values for matchingMode:
                        
    
                            
  • 'contains'
    • 
                            
  • 'prefix'
    • 
                            
  • 'prefix-word'
    • 
                            
  • 'prefix-levenshtein' (experimental)
    • 
                            
  • 'levenshtein' (experimental)
    • 
                        

                    		
                    {matchingMode: 'contains', ignoreCase: true, maxLevenshteinDistance: 2}
